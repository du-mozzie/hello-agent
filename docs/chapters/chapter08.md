# Chapter 08 - Memory and Retrieval

## Goal

Implement working memory, episodic memory, vector retrieval, memory consolidation, and a small RAG pipeline.

## Implemented Code

- `WorkingMemory`
- `ConversationMemory`
- `EpisodicMemory`
- `VectorMemory`
- `MemoryConsolidator`
- `RAGPipeline`
- `RAGQAAssistant`

## Run

```powershell
python examples/chapter08_memory_rag.py
```

## Extension

Swap `VectorMemory` for a real vector database and add document loaders for PDF, HTML, Markdown, and databases.
