# 第 14 章：自动化深度研究智能体

## 学习目标

构建一个包含规划、检索、证据整理和综合写作的研究型 Agent。

## 对应实现

- `DeepResearchAgent`
- `ResearchReport`
- `VectorMemory`

## 运行示例

```powershell
python examples/chapter14_deep_research.py
```

## 代码要点

深度研究不是单次问答，而是多阶段流程：定义范围、收集证据、比较取舍、综合结论。

## 扩展方向

加入 Web 搜索、引用抽取、来源排序、报告修订、Markdown/PDF/HTML 导出。
