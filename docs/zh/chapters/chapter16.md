# 第 16 章：毕业设计

## 学习目标

把工具、Agent、记忆、上下文、评估和应用组合成一个完整项目。

## 对应实现

- `ReActAgent`
- `Evaluator`
- `TravelPlanner`
- `DeepResearchAgent`

## 运行示例

```powershell
python examples/chapter16_capstone.py
```

## 代码要点

毕业设计不应只是一个 Demo，而应包含可运行代码、模块边界、文档、测试和评估。

## 扩展方向

继续加入前端界面、持久化存储、用户系统、部署脚本、监控日志和回归评测集。
